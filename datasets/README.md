Code is ready to run with the George Washington dataset. This can be downloaded from http://www.cvc.uab.es/~almazan/data/GW.tar.gz

Alternatively, you can also download other great datasets that have been tested:

IAM: http://www.iam.unibe.ch/fki/databases/iam-handwriting-database
IIIT 5K-Word: http://cvit.iiit.ac.in/projects/SceneTextUnderstanding/IIIT5K.html
Stree View Text: http://vision.ucsd.edu/~kai/svt/
